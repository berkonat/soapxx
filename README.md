# soapxx

### Installation
```bash
git clone https://github.com/capoe/soapxx.git
cd soapxx
./build.sh
source soap/SOAPRC
```

### Dependencies
- boost (python mpi filesystem serialization) (Version < 1.64)
- openmpi
- GSL (Version 2.1) or MKL (MKL + intel compilers are recommended)
- GCC 5.4.1 Compilers (Version < 7)
- Python < 2.7
